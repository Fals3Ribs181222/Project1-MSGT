Bulk Operations

Destructive operations. All actions require confirmation and are permanent.

Delete All Attendance for a Batch

Permanently removes every attendance record belonging to the selected batch.

Clear All Marks for a Test

Deletes every mark record for the selected test. The test itself remains.

Delete a Student Account

Permanently deletes the student's auth account and all related data (marks, attendance, batch memberships).

Wipe All Seed Data

Deletes all student accounts and their linked data (marks, attendance, batch memberships). Teacher accounts are preserved. This cannot be undone.
